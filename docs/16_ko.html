Struct and Enum

In Rust, struct and enum are two ways to define custom data types. A struct (short for "structure") is a composite data type that groups related values together. An enum (short for "enumeration") is a data type that consists of a fixed set of named values, called "variants."

Struct

A struct is defined using the struct keyword, followed by a name and a set of named fields enclosed in curly braces. Here's an example of a Person struct: rust struct Person { name: String, age: u8, } To create an instance of a struct, use the struct name followed by curly braces and assign values to its fields. For example:

rust let person = Person { name: String::from("Alice"), age: 30 };

Enum

An enum (short for "enumeration") represents a type that has a fixed set of named values, called "variants". To define an enum, use the enum keyword followed by a name and a set of named variants enclosed in curly braces. Here's an example of a Status enum: rust enum Status { Active, Inactive, Pending, } To create an instance of an enum, use the enum name followed by two colons (::) and the variant name. For example: rust let status = Status::Active; You can use match expressions to perform different actions based on the enum variant: rust let user_status = Status::Active; To work with enums, we often use match expressions, which allow us to perform different actions depending on the enum variant: rust fn print_status_message(status: Status) { match status { Status::Active => println!("User is active."), Status::Inactive => println!("User is inactive."), Status::Pending => println!("User is pending."), } } fn main() { let user_status = Status::Active; print_status_message(user_status); }
