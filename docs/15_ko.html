match

The match keyword in Rust is used for pattern matching.

It allows you to compare a value against a series of patterns

and execute a block of code corresponding to the first matching pattern. ```rust fn main() { let number = 42;

match number {
    0 => println!("The number is zero"),
    1 => println!("The number is one"),
    42 => println!("The answer to life, the universe, and everything"),
    _ => println!("The number is something else"),
}

} we match the value of the number variable against several patterns. The _ pattern is a catch-all pattern that matches any value not explicitly covered by previous patterns. rust fn classify_age(age: u8) { match age { 0..=12 => println!("Child"), 13..=19 => println!("Teenager"), 20..=64 => println!("Adult"), _ => println!("Senior"), } } fn main() { let age = 25; classify_age(age); } ``` we use a match expression to classify ages into groups. We use ranges as patterns to match against the age variable.
